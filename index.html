JavaScript com cartas

Conceitos simples para quem sente dificuldade em aprender.

Nunca desista dos seus sonhos!

Variáveis

Existe em JavaScript 3 meios de declara-las

Var, let ou const!

Podemos declarar assim:

var exemplo = "Este Exemplo"

Funções

Podemos criar funções utilizando a palavra reservada "function" e logo após o nome da sua função

após isto, utilizamos () e chaves.

function x () {}

Arrays

Arrays é uma forma de criar listas

Utilizamos colchetes e virgulas a cada item da lista, exemplo:

["maça", "banana"]

Objetos

Tudo em javascript é objeto!

É uma coleção de dados, onde pode-se ter

propriedades ou metodos

Metodos

De forma bem resumida e simplificada

Metodos são ações em javascript, como se fossem verbos!

Podemos cria-los, ou usar metodos integrados.

Propriedades

Imagine uma pessoa

Propriedades são as caracteristicas dessa pessoa!

cor dos olhos, cabelos e afins..

Dom

Dom é uma forma de manipular coisas..

Imagine que você queira pegar um ID no JS

Podemos pegar esse ID usando um metodo da dom chamado

getElementByID

Classes

classe é como uma

Fabrica de objetos

Strings

Strings são como as palavras normais, elas servem para representar letras

Podemos utilizar uma string usando ""
Um exemplo pratico seria: var nome = "Tawan"

Booleanos

Booleanos são valores de sim ou não.

Imagine uma lanterna, você pode ligar ou desligar..

Em javaScript, isso é representado como true ou false.

if e else

if é "se" e else é "se não.."

Uma pessoa só pode comprar SE tiver dinheiro, SE NÃO, ela não pode comprar.

Concatenação

Podemos juntar variaveis com palavras, usando o sinal de +

Imagine que temos uma variavel chamada nome, então fariamos:

"Meu nome é" + nome
